# TNS E-message Project

A small project to extract collected data from excel (custom formatted) into a well readable format and send the required information to the intended destination.

## Basic Requirements

- Extract the data correctly ❌
- Store the data in JSON file ❌
- Obtain a reputable SMS gateway's API ❌
- Send the message and see its state of delivery at any time ❌
- Deliver the message to different people, meaning that smart phone messages (long) and non-smart message (short) ❌
- Have records of people and status of their message

## Additional Requirements

- Have a GUI for easier usage
- Integrate it to excel
